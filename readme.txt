== Description ==

iMasters WP FAQ is used to manage FAQ in your WordPress based site. Organize your FAQ's in categories and put them in posts or pages.

== Installation ==

How to install iMasters WP FAQ

e.g.

1. Upload `imasters-wp-faq` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP FAQ' menu in WordPress
